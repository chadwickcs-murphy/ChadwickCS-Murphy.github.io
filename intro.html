Introduction to Coding

Course Description

This course is designed for students that have minimal programming experience but are looking for a chance to learn more about programming. Students will mainly create graphical applications while they learn the basics of Java. This course is also designed to prepare students for AP Computer Science A.

  • strokeWeight(weight)
  • noStroke()
  • circle(centerX, centerY, diameter)
  • square(upperLeftX, upperLeftY, sideLength)
  • rect(upperLeftX, upperLeftY, width, height) or rect(upperLeftX, upperLeftY, width, height, cornerRadius)
  • triangle(x1, y1, x2, y2, x3, y3)
