AP Computer Science A

Course Description

The course is designed for 10th-grade through 12th-grade students with previous programming experience who want to further their computer programming skills in the Java programming language. Students interested in taking the course should reach out to the teacher to determine if their prior background is sufficient. Students learn the basics of object-oriented programming in this course and practice utilizing input/output, conditionals, loops, and data structures to build applications. This course prepares students to take the AP Computer Science A exam in May.
